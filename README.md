# Password Strength Checker

This project is a simple Python security tool that evaluates password strength based on common cybersecurity best practices.

## Features

- Checks password length
- Detects uppercase and lowercase characters
- Detects numbers
- Detects special characters

## Example

Enter a password to test:

MyPassword123!

Output:

Password strength: Strong Password

## Purpose

This project was built to practice Python scripting and demonstrate understanding of basic security concepts related to password strength and authentication.
